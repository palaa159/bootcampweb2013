Introducing jQuery

jQuery is a library written in JavaScript (not Java, totally different from Java in Processing !).

Library" is a package that some code-genius has developed to serve a certain purpose or function, usually to help facilitate developers and designers from figuring out how to achieve that purpose.

It is crucial to take a look at the library's document. Whatever library you're using, even the jQuery extensions, the documentation is always provided in their website or in the GitHub repo. For example, this is the jQuery Documentation and the GitHub Repo. Just ignore what's happening in GitHub for now.

Syntax

$('#forID').function();

$('.forClass').function();
